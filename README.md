# Predicci-n-Inteligente-de-Precios-de-Viviendas-mediante-Redes-Neuronales-Profundas
Predicción de precios de viviendas mediante redes neuronales profundas utilizando TensorFlow, Scikit-Learn y el California Housing Dataset.

## Características

- Análisis exploratorio de datos (EDA)
- Comparación de múltiples arquitecturas neuronales
- Evaluación mediante RMSE, MAE y R²
- Comparación contra Regresión Lineal
- Interfaz interactiva desarrollada con Gradio
- Entrenamiento reproducible con TensorFlow/Keras

## Tecnologías

- Python
- TensorFlow / Keras
- Scikit-Learn
- Pandas
- NumPy
- Matplotlib
- Gradio

## Dataset

California Housing Dataset disponible a través de Scikit-Learn.

## Resultados

El mejor modelo obtuvo:

| Métrica | Valor |
|----------|----------|
| RMSE | 0.5331 |
| MAE | 0.3674 |
| R² | 0.7831 |
